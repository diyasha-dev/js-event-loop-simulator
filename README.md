# JavaScript Event Loop Simulator

A browser-based visualization tool that shows how JavaScript handles 
async code — including the Call Stack, Microtask Queue (Promises), 
and Macrotask Queue (setTimeout).

## What It Does
- Visualizes JavaScript's concurrency model live
- Step-through execution mode
- Adjustable animation speed
- Preset code examples (basic, promises, async/await)
- Live output console

## Tech Stack
HTML5 · CSS3 · Vanilla JavaScript

## How to Run Locally
Just open index.html in your browser — no install needed.
